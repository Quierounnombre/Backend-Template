## Template of Backend in Golang

A generic template for creating a stable, secure and performant backend in mere minutes.

It uses a yaml for non critical data, and a .env for critical data(KEEP THIS OUT OF THE PROD REPO)

1. OAuth support with Google
2. Pgxpool configured
3. User creation configured
4. Secure implementation of auth and routing
5. Configuration made with yaml
6. JWT

Easy and simple to just create backends by the kilo.
